## Features

-   人気やオッズ、その他**第三者視点での評価を徹底的に排したデータ至上主義のサービス**を提供
-   中央・地方を問わず、毎日最新のデータを更新
-   みやすいデータ可視化（とインタラクティブな GUI（実装中））
-   改善点を広く募集中 🔥
